# [PROTOCOLO DE CONSISTÊNCIA VISUAL - MrTok]

Este documento define as regras de "DNA" para evitar o desperdício de créditos em vídeos alucinados. Todo vídeo gerado deve obrigatoriamente seguir este pipeline de 3 camadas.

## 1. Camada de Identidade (Character DNA)
- [cite_start]**O que é:** Uma folha de referência (Character Sheet) 360º gerada pelo Nano Banana 2.
- [cite_start]**Regra:** Proibido iniciar a geração de cenas sem antes ter o arquivo `dna_influencer.png` aprovado pelo QC[cite: 114].
- [cite_start]**Consistência:** O prompt das cenas deve sempre referenciar o DNA (ex: "Using Image 1 as reference, show the same character...")[cite: 195].

## 2. Camada de Ambiente (Environment DNA)
- [cite_start]**O que é:** Uma imagem limpa do cenário (ex: a cozinha, o quarto, o escritório) sem personagens[cite: 81, 101].
- **Objetivo:** Travar a iluminação e as cores de fundo. [cite_start]Se a luz mudar no meio do vídeo de 40s, a retenção cai[cite: 125].

## 3. Camada de Storyboard (A Física do Movimento)
- [cite_start]**O que é:** Um grid 3x3 que mostra a evolução da ação em 9 frames[cite: 119, 163].
- [cite_start]**Uso Técnico:** Serve para extrair o `Start Frame` e o `End Frame` de cada bloco de 10 segundos[cite: 120, 168].
- [cite_start]**O Pulo do Gato:** O último frame do Vídeo 1 (SH1B) DEVE ser usado como referência visual para o primeiro frame do Vídeo 2 (SH2A)[cite: 143].
