# /workspace/video-renderer

Projeto **Remotion** do Agente 6 (Montador CLI). Aplica a estratégia **Unique Pixel Hash** (micro-variações aleatórias de escala 1.01x e rotação 0.1º) a cada export para garantir hash única e driblar detecção de conteúdo não-original do TikTok Shop.

> Esqueleto apenas na Tarefa 1. Implementação completa do Remotion + Unique Pixel Hash ocorre na **Tarefa 3**.
